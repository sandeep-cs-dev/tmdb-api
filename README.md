# tmdb-api

### Project setup

    $ git clone https://github.com/sandeep-cs-dev/tmdb-api.git
    $ cd tmdb-api
    $ install required dependencies using npm install command

## Configure app (Development)

Create .env file under project root directory

- PORT='PORT'
- TMDB_API_KEY='put your key here'
- TMDB_BASE_URL=http://api.themoviedb.org/3

## Running the project in dev mode
  $ npm run dev
  
## Running the test
  $ npm run test-dev
  
## compile
  $ npm run build  

## start server
  $ npm start  

## API End points
- /topEpisodes/{seriesId}
- /analytics/popularSeries

## Note
Check Mock-data folder for sample seriesIds
